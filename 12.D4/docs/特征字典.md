# D4 时域+频域+跨传感器融合特征字典
## 一、基础说明
本项目单窗口时序维度：256采样点，共18路通道（9轴原始IMU信号 + 9轴高低通分离重力/运动分量）
每一路通道固定提取 **12维时域特征 + 6维频域特征**；额外增加4维跨传感器融合特征。
代码支持一键开关特征组，用于消融对比实验。

## 二、单通道时域特征（12个，B层基础任务要求）
| 特征名称 | 数学计算公式 | 物理工程含义 |
| ---- | ---- | ---- |
| 均值 | $\mu=\frac{1}{N}\sum_{i=1}^N x_i$ | 信号直流偏移量，区分静坐、站立两类静态姿态基线 |
| 方差 | $\sigma^2=\frac{1}{N}\sum_{i=1}^N (x_i-\mu)^2$ | 运动波动剧烈程度，跑步/下楼方差远高于静坐 |
| 标准差 | $\sigma=\sqrt{Var(x)}$ | 信号波动幅值，直观反映肢体活动幅度 |
| 最大值 | $\max(x)$ | 动作冲击峰值，如踏步落地、抬腿极值 |
| 最小值 | $\min(x)$ | 运动谷底幅值，对应肢体回落最低点 |
| 峰峰值 | $\max(x)-\min(x)$ | 单次完整动作的整体波动区间 |
| 中位数 | 时序序列排序中间值 | 抗传感器突变尖峰干扰，稳定表征静态偏移 |
| RMS均方根 | $\sqrt{\frac{1}{N}\sum x_i^2}$ | 信号总能量，同时包含静态直流与动态运动分量 |
| 偏度 | $E[(x-\mu)^3]/\sigma^3$ | 分布对称度；上楼正向偏置、下楼负向偏置 |
| 峰度 | $E[(x-\mu)^4]/\sigma^4 - 3$ | 信号尖峰锐利程度，跑步冲击动作峰度更高 |
| 过零率 | 穿过均值的采样点总数 / 总采样数 | 振荡频繁程度：慢跑 > 步行 > 静坐站立 |
| SMA信号幅值面积 | $\frac{1}{N}\sum\|x_i\|$ | 单通道平均运动强度，用于区分动静状态 |

## 三、单通道频域特征（6个，衔接传感器PSD频谱知识点）
1. **主频FFT**
频谱幅值最大对应的频率，代表人体基础步频；步行约2Hz，慢跑约3Hz。

2. **频谱质心**
频谱加权重心，数值越高代表整体运动频率越快。

3. **谱熵**
频谱杂乱程度；规律匀速行走熵值低，随意无规则活动熵值高。

4. **0.5~3Hz频带能量比**
人体正常步行、慢跑核心运动频段能量占全部频谱的比例，滤除高频噪声干扰。

5. **谱峰数量**
动作谐波丰富度，复合动作（上下楼）谱峰多于简单行走。

6. **频谱带宽**
有效运动信号覆盖的频率区间宽度。

## 四、A层拓展：跨传感器融合特征（4维）
1. 加速度合模均值：消除传感器安装朝向影响，仅反映三轴总运动量；
2. 陀螺仪合模标准差：肢体旋转、转弯抖动剧烈程度；
3. ax-gx相关系数：平动加速度与旋转角速度的运动耦合强弱；
4. 磁场模方差：磁力计三轴模长波动，表征人体转向、航向变化幅度。

## 五、S层可配置流水线消融实验说明
代码头部布尔变量可一键启用/关闭整组特征，控制变量对比模型精度：
1. `USE_TIME_FEATURE = False`：仅保留频域特征训练，对比时域特征贡献；
2. `USE_FREQ_FEATURE = False`：仅保留时域特征训练，对比频域特征贡献；
3. `USE_FUSION_FEATURE = False`：关闭多传感器融合特征，验证融合特征带来的识别精度增益。

## 六、特征矩阵输出说明
代码输出 `feature_matrix.csv`，列命名规则：
- `通道X_均值`：第X路通道的均值时域特征
- `通道X_主频`：第X路通道的主频频域特征
- `融合_加速度合模均值`：全局跨传感器融合特征
末尾两列为附加标签：`标签数值`（0~5）、`活动类别`（sit/stand/walk/jog/upstairs/downstairs）
